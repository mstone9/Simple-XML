# Simple-XML
This repository is used to teach future developers on how to use Simple framework, using three seperate use cases.
## This repository will teach developers how to:
- [Add Simple XML as a maven dependency](https://github.com/mstone9/Simple-XML/wiki#how-to-add-simple-framework-to-a-maven-project)
- [Creating and using the persister](https://github.com/mstone9/Simple-XML/wiki#the-persister)
- [Use Simple XML to parse a basic XML file](https://github.com/mstone9/Simple-XML/wiki/Parsing-A-Basic-XML-File)
- [Use Simple XML to parse a nesting XML file](https://github.com/mstone9/Simple-XML/wiki/Parsing-a-Nested-XML-File)
- [Use Simple XML to parse an array expressed as an XML file](https://github.com/mstone9/Simple-XML/wiki/Parsing-an-XML-File-using-an-Array)
